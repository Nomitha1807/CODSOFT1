#  Manthena Nomitha Varma - Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This website showcases my skills, projects, resume, and contact information with a clean pastel-themed user interface.

## 🔗 Live Demo

Visit the website here:

**https://nomitha1807.github.io/CODSOFT1/**

---

## 📌 Features

* Responsive design for mobile, tablet, and desktop devices
* Smooth scrolling navigation
* Animated pastel gradient background
* Professional hero section with profile image
* Skills section with progress bars
* Projects showcase section
* Resume download functionality
* Contact section with email link
* Modern glassmorphism-inspired design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Poppins)
